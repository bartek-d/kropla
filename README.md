# kropla
Kropki game engine

# Building

Clone the repo
```
git clone git@github.com:bartek-d/kropla.git
cd kropla
```

Init and fetch submodules
```
git submodule init
git submodule update
```

Build
```
mkdir build
cd build
cmake ..
make kropla
```

# Running
```
./kropla
```
runs the program in an interactive mode.
